2. 优秀项目组织方式
   * checkpoints: 模型训练断点保存
   * data: 数据集，数据集处理和相关实现代码
   * models: **模型定义，相同功能部分需要抽象成为抽象类准备继承(比如保存和加载模型参数的方法)**
   * utils: 工具函数，可视化工具等等
   * config.py: 配置文件
   * main.py: 主文件
